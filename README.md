# Ordersounds Desk Prototype

Standalone copy of the Ordersounds Desk prototype.

## Files

- `src/pages/AiLabelPrototype.tsx` - prototype screen and flow
- `docs/ai-record-label-prd.md` - product requirements document
- `docs/google-stitch-redesign-handoff.md` - design handoff aligned to the current prototype

## Run locally

```bash
npm install
npm run dev
```

The current local prototype is running at `http://127.0.0.1:5173/`.
